  • First, let's understand that code has standards. PEP 8 standards, to be exact. PEP 8 is the correct format in which to write code. You don't know what that correct format is so you would use the following code in your terminal to have Python tell you how your code should be written, what's missing, where to add the missing information and if your code is overall erroneous:

    • pytest --pylint --cov

  • 1966 – Bob Taylor, ARPA Scientist needed a separate terminal to log onto each time-shared computer across the country. Since no 2 computers could communicate with each other, Taylor had to move through each terminal. He figured a single terminal would be better if it could communicate with all the computers at once instead through one network.
  • Taylor asked for money to build a computer network – “resource sharing” to save time and money for ARPA.
  • Taylor recruited Dr. Lawrence (Larry) Roberts, to help with his project. Roberts had built the world’s first experimental connections between 2 computers at MIT.
  • Robert had to figure out how to network main frame computers.
  • An engineer inspired Larry by suggesting building identical special purposes mini computers to form the network and tie each time-shared computer to them.
